Mountain Madness

Bob loves golfing.

Specifically, he loves minigolfing. Recently, he had a dream idea for a minigolf hole! He wants to make a hole at the top of an alphabet mountain.

He's asked you to make this mountain and hole for him. But in true minigolf fashion, he's asked that your code to do so is as small as possible.

Input

The input will consist of a single integer $n$ ($1 \leq n \leq 27$), the size of the mountain.

Output

Output the mountain, hole, and flag, as in the samples. Please be careful with whitespace, as the solution is sensitive to whitespace used.

Sample Input 1

3

Sample Output 1

    >
    |
    .
   /A\
  /ABA\

Sample Input 2

1

Sample Output 2

>
|
.

Sample Input 3

5

Sample Output 3

    >
    |
    .
   /A\
  /ABA\
 /ABCBA\
/ABCDCBA\

Scoring

You will get 50 points for submitting a working solution.

The length of your code will be counted in number of ASCII characters. Furthermore, for each language, points will be assigned as follows:
- Shortest code - 100 points
- 2nd shortest code - 83 points
- 3rd shortest code - 67 points
You can only get points for one language, and the language in which you scored the highest will give you points. This means that you can deny teams points by scoring well in multiple languages.
